
# backend-exchange-api-test

Este el Exchange API 

## Caracteristicas

- NestJS
- JWT (Para autenticacion)
- Redis (para almacenamiento)



## Instalacion

```bash
$ npm install
```

## Ejecutar

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
